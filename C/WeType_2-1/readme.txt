                      "WeType" 게임방법

 1-1. '2. 추가' : 사용자를 추가한다. (최대 20명까지 추가 가능, 이름은 최대 20자)

 1-2. '1. 선택' : 사용자를 선택한 후 게임의 난이도를 선택한다.
  (easy: 4글자 6초, normal: 6글자 5초, hard: 8글자 4초 제한)

 1-3. 각 난이도는 15문제로 구성되며 80%(12문제) 이상 정답을 맞히면 다음 난이도로 넘어간다.
  (단 hard 난이도에서는 해당 난이도 반복 혹은 게임 종료를 선택할 수 있다.)

 1-4. 1)난이도별 정답 시 추가 점수:
   Easy: 기본 50 + (남은 시간 x10)점
   Normal: 기본 70 + (남은 시간 x15)점
   Easy: 기본 100 + (남은 시간 x20)점
      2)난이도별 오답 시 차감 점수:
   Easy: -50점, Normal: -70점, Hard: -100점

 1-5. 게임을 실행하면 상단에 현재 난이도, 현재 점수, 정답률, 분당 타수가 출력된다.

 1-6. 숫자 1을 눌러 게임을 잠시 정지시키거나 이어서 시작할 수 있다.

 1-7. 게임 중 esc를 눌러 메인 메뉴로 돌아갈 수 있다.

 2-1. '3. 조회' : 사용자별로 난이도별 누적 점수, 정답률, 분당 타수를 확인한다.

 2-2. '4. 수정', '5. 삭제' : 사용자 이름을 수정하거나 사용자를 삭제한다.

 2-3. '6. 순위표' : 난이도 별 사용자의 순위를 확인한다.

 2-4. '8. 크레딧' : 제작 방향 및 기획 의도를 확인한다.

 2-5. '0. 종료' : 게임을 종료한다.
